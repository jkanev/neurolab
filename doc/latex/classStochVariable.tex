\subsection{Detailed Description}
A simple stochastic variable. 

This is the simplest stochastic process available. All stochastic processes which are not parametrised at every time step should inherit from this class. If you derive from this class, you must provide a \doxyref{next()}{p.}{classStochastic_eabb113d791b85414440801f20f444a1} function, which sets dNextValue by using dCurrentValue. \subsection*{Public Member Functions}
